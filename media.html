Média

A média (Me) é uma medida estatística importante que representa o valor médio de um conjunto de números. Ela é calculada somando todos os valores e dividindo pelo número total de elementos. A média aritmética é frequentemente usada para resumir um conjunto de dados e fornecer uma ideia geral de seu valor central. Para calcular a média aritmética de um conjunto de números, você deve somar todos os valores e dividir pelo número total de elementos. Por exemplo, se tivermos os seguintes números: 5, 7, 10, 12 e 15, a média aritmética seria calculada da seguinte forma:
(5 + 7 + 10 + 12 + 15) / 5 = 49 / 5 = 9,8
Portanto, a média aritmética dos números fornecidos é 9,8.

Fórmula

Sendo, Me: média x1, x2, x3,..., xn: valores dos dados n: número de elementos do conjunto de dados

Exemplo

Os jogadores de uma equipe de basquete apresentam as seguintes idades: 28, 27, 19, 23 e 21 anos. Qual a média de idade desta equipe?
